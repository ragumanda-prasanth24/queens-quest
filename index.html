N-Queens Problem Rules

No Two Queens in the Same Row:

Only one queen is allowed per row. If a block in a row already has a queen, clicking on another block in the same row should remove the first queen.

No Two Queens in the Same Column:

If placing a queen in a column where another queen exists, it is invalid. Clicking on such a block should either toggle off the queen or display a warning.

No Two Queens on the Same Diagonal:

Diagonal constraints must be maintained. For major diagonal (row - col) and minor diagonal (row + col), ensure no queens conflict. Toggling a block that violates these constraints should either do nothing or notify the user.

Interactive Toggle:

Allow users to click on any block to place or remove a queen, provided no rules are violated.
